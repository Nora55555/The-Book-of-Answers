# PRD：答案之书 网页小游戏

## 产品概述

一个面向小红书用户的治愈系互动网页小游戏，模拟经典《答案之书》的翻阅体验。用户在心里默念问题后翻开书页，获得一个随机的答案指引。

## 核心价值

- **情绪价值**：为迷茫/犹豫的用户提供温柔的心理暗示
- **仪式感**：翻书动效 + 冥想等待 + 答案浮现，构建完整仪式体验
- **分享欲**：精美分享卡片激发用户在小红书发布

## 用户故事

1. 用户打开页面 → 看到精美书本封面 → 点击"翻开答案"
2. 封面3D翻页 → 进入书本内页 → 引导用户默念问题
3. 3秒冥想等待（带呼吸光效）→ 答案自动浮现
4. 用户看到答案 → 可选择再问一次 / 查看历史 / 保存分享

## 功能清单

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 3D翻书动画 | 封面翻开、页面前翻 | P0 |
| 答案随机生成 | 60+条答案，4大类型 | P0 |
| 冥想等待 | 3秒呼吸光效，可跳过 | P0 |
| 墨迹答案动效 | 答案浮现动画 | P0 |
| 历史记录 | localStorage保存，可查看/清空 | P1 |
| 分享卡片 | Canvas生成精美卡片，可保存 | P1 |
| 粒子系统 | 环境粒子漂浮 | P1 |
| 响应式适配 | 手机/平板/桌面 | P0 |

## 非功能需求

- 零依赖，纯静态页面
- 首次加载 < 2秒
- 无任何账号/数据收集
- 移动端触摸友好
- 支持 iOS Safari / Android Chrome / 微信内置浏览器

## 技术栈

HTML5 + CSS3 + Vanilla JavaScript (无框架)
