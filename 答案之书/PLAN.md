# 答案之书 · 网页小游戏 设计方案

## 产品定位

面向小红书用户的治愈系互动网页小游戏，模拟经典《答案之书》的翻阅体验。极简交互、顶级视觉、丝滑动效，无需注册即开即玩。

---

## 目标用户画像

| 维度 | 描述 |
|------|------|
| 年龄 | 18-35 岁 |
| 性别 | 以女性为主（小红书核心用户），文案中性不排他 |
| 特征 | 追求生活仪式感、喜欢治愈/玄学内容、乐于分享截图 |
| 使用场景 | 深夜emo、选择困难、考前/面试前求心安、日常小确幸 |

---

## 小红书雷点规避清单

| 雷点 | 解决方案 |
|------|----------|
| ❌ UI土气/过时 | ✅ 奶油色系 + 毛玻璃 + 极简高级感 |
| ❌ 需要登录注册 | ✅ 零账号体系，打开即玩 |
| ❌ 广告/弹窗 | ✅ 纯公益小游戏，零广告 |
| ❌ 加载慢 | ✅ 纯静态页面，总大小 <300KB（不含字体） |
| ❌ 操作复杂 | ✅ 仅需点击/触摸，1秒上手 |
| ❌ 抄袭感/千篇一律 | ✅ 原创3D翻书动效 + 50+条独家文案 |
| ❌ 性别/隐私暴露 | ✅ 文案中性，不收集任何数据 |
| ❌ 低质动画卡顿 | ✅ CSS硬件加速 + requestAnimationFrame |

## UI/UX 设计

### 视觉风格

- **配色方案**：
  - 背景奶油白 `#FBF6F0`
  - 纸张暖白 `#F5EDE3`
  - 书脊/文字深棕 `#3C2415`
  - 点缀金色 `#C9A96E`
  - 阴影暖灰 `rgba(60,36,21,0.1)`
- **字体**：`Noto Serif SC`（正文/答案）+ `system-ui`（UI文字）
- **设计语言**：模拟精装典籍质感，圆角柔和，光影自然
- **氛围**：温暖、安静、治愈、有仪式感

### 交互流程

```
┌─────────────┐
│   首页封面    │  ← 精装书封面，带环境光效
│  "答案之书"   │
│  [打开]       │
└──────┬──────┘
       │ 点击 → 3D翻页动画（封面打开）
       ▼
┌─────────────┐
│  冥想引导页   │  ← "在心里默念你的问题……"
│  (呼吸光效)   │  ← 3秒等待，增强仪式感
│  ● ● ●       │
└──────┬──────┘
       │ 自动过渡 → 书页翻开
       ▼
┌─────────────┐
│  答案展示页   │  ← 手写体答案浮现在右侧书页
│  "大胆去做"   │  ← 墨迹扩散动效
│              │
│ [再问一次]    │
│ [分享] [记录] │
└─────────────┘
```

### 页面状态

1. **首页**：闭合书本居中，微浮动效，背景暖色渐变，底部"打开答案之书"按钮
2. **翻页动画**：3D CSS transform 模拟真实翻书，约1.2秒
3. **冥想页**：书本打开状态，左侧书页显示提示文字，右侧空白带呼吸光晕，3秒倒计时
4. **答案页**：答案墨迹浮现，伴有细微粒子飘散，底部操作栏
5. **历史记录弹窗**：半透明毛玻璃浮层，展示过往答案列表

## 答案数据库

50+条原创答案，分为4大类：

| 类型 | 比例 | 示例 |
|------|------|------|
| 肯定型 | 30% | 是、大胆去做、时机已到、一切皆有可能、放手一搏 |
| 否定型 | 20% | 再等等、不是现在、放下执念、换个方向 |
| 指引型 | 30% | 听从内心、换个角度、向前走、相信直觉 |
| 开放式 | 20% | 答案在你心里、随缘、顺其自然、时间会告诉你 |

## 技术方案

- **架构**：纯前端 SPA，零框架零依赖
- **翻书引擎**：CSS 3D `transform: rotateY()` + `perspective`
- **粒子效果**：Canvas 2D 绘制的微尘/光点粒子
- **墨迹动效**：CSS `clip-path` 遮罩动画模拟墨水扩散
- **状态管理**：localStorage 存储答案历史
- **响应式**：移动端优先，适配 320px ~ 1920px
- **性能**：所有动画 GPU 加速，无布局抖动
